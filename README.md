###  What is the difference between var, let, and const?
var was once used in js to declare variables, because of hoisting now it's not used anymore.

let is modern es6 variable declaration system which is reassignable.

const is modern es6 variable declaration system which is not reassignable.


###  What is the spread operator (...)?
spread operator  copies  elements from an array or object into another array or object.


###  What is the difference between map(), filter(), and forEach()?
map()– creates a new array by changing each item in the original array.
filter() – creates a new array with only the items that pass a condition.
forEach() – runs a function on each item and does not return an array.



###  What is an arrow function?
Arrow function is a short way to write a function in JavaScript using => sign.


###  What are template literals?
Template literals is a way to write strings in JavaScript using backticks  that allow to write multi-line text.
